# 🚀 IA Mentor de Carreira

Este projeto faz parte do bootcamp **Caixa - Inteligência Artificial na Prática** da [DIO](https://www.dio.me).  
O objetivo é criar um **agente de IA** que funcione como um mentor de carreira, ajudando pessoas a descobrirem seu futuro em tecnologia.

## 📌 Objetivo

- Coletar informações do usuário sobre carreira, experiência e interesses.  
- Analisar os dados e sugerir áreas de atuação em tecnologia.  
- Gerar um **plano de estudos de 90 dias** com roadmap, projeto de portfólio e roteiro de entrevistas.  
- Indicar trilhas de aprendizado na DIO para apoiar o desenvolvimento.

---

## 🛠 Tecnologias utilizadas

- **GitHub** → versionamento e documentação  
- **Copilot** → suporte na criação do agente de IA  
- **Prompt Engineering** → estruturação das interações  
- **Markdown** → documentação clara e organizada  

---

## 📂 Estrutura do projeto

- **Agente 1** → coleta informações do usuário  
- **Agente 2** → gera roadmap completo e personalizado  
- **Feedback Agente1** → análise inicial das respostas  
- **Feedback Agente2** → plano detalhado de carreira  
- **examples/** → exemplos de roadmaps gerados (ex.: QA Jr, Front-End, Dados)
