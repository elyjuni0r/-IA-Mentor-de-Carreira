# 🚀 IA Mentor de Carreira

Este projeto faz parte do bootcamp **Caixa - Inteligência Artificial na Prática** da [DIO](https://www.dio.me).  
O objetivo é criar um **agente de IA** que funcione como um mentor de carreira, ajudando pessoas a descobrirem seu futuro em tecnologia.

---

## 📌 Objetivo
Desenvolver um agente de IA capaz de:
- Fazer perguntas sobre interesses e habilidades do usuário.
- Analisar as respostas e sugerir áreas de atuação em tecnologia (ex.: Front-end, Back-end, Dados, IA, Cloud).
- Gerar um **roteiro de estudos personalizado** com base no perfil identificado.

---

## 🛠 Tecnologias utilizadas
- **GitHub** → Para versionamento e publicação do material
- **Copilot** → Para criar o agente de IA.
